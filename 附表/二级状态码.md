# errCode 二级状态码

| **值** | **含义** |
| :--- | :--- |
| 0 | 正常结果 |
| 1 | 自动捕捉的错误 |
| 15 | 参数校验错误 |
| 21 | 用户密码验证相关错误 |
| 22 | 未找到代理代理服务器错误|
| 23 | 请求超时错误|
| 28 | jtw错误 |
| 29 | 没有权限,被拒绝 |
| 30 | 未认证的请求,一般指身份认证失败,需要登录 |
| 100 | 异常占位区 |
| 70080101 | 未找到用户合同 |
| 70080102 | 选择需要执行的合同.一般有两个或两个以上的时候需要用户选择具体执行哪个 |
| 70080103 | 参数userContractId错误,找不到有效的合同 |
| 70080104 | 用户的合同未激活 |
| 70080105 | 用户合同授权异常 |
| 70080201 | 未找到节点与资源的合同信息(除非DB错误,一般不会出现)  |
| 70080202 | 节点的合同未激活 |
| 70080203 | 节点合同授权异常 |


** 说明: 具体应用层返回的错误 **
