# 授权服务状态码

| **authCode值** | **authCode值含义** |
| :--- | :--- |
| 1 | 基于用户合约授权 |
| 2 | 基于节点合同授权 |
| 3 | 基于节点的策略授权 |
| 4 | 基于资源的策略授权 |
| 5 | 基于个人用户认证 |
| 6 | 基于用户组策略 |
| 7 | 基于用户组或用户组策略 |
| 11| 用户合同不满足 |
| 12| 节点合同不满足 |
| 13| 节点策略不满足 |
| 14|资源策略(resource-policy)不满足 |
| 15|消费策略(presentable-policy)不满足|
| 16|目标用户不满足uses策略|
|100|未知的异常,error|
