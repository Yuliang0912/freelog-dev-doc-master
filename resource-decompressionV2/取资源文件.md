#取资源文件

### 调用方式:GET

### 接口地址:

```
http://api.freelog.com/v2/resource-decompressions/getResourceFile
```

### url传入参数说明:

| 参数    | 必选 | 类型及范围 | 说明           |
| :------ | :--- | :--------- | :------------- |
| name    | 必选 | string     | 根文件夹名称   |
| version | 必选 | string     | 目标文件版本号 |
| tarPath | 必选 | string     | 目标文件名称   |
