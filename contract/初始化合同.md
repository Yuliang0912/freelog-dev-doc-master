# 初始化合同

### 调用方式: GET

### 接口地址:

```
https://api.freelog.com/v1/contracts/initial?contractIds={contractIds}
```

### url传入参数说明：

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
|contractIds|必选|string|合同ID,多个用逗号分隔.支持范围1-100个|


### 返回说明(只会返回有权限操作,并且是初始态的合同)：
| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
|  contractId | string | 合同ID

```js
{
    "ret": 0,
    "errcode": 0,
    "msg": "success",
    "data": [{"contractId":"59cb56f9ea4b651a4c69ff1f"}]
}
```
