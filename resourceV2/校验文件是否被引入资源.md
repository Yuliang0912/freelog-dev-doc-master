# 校验文件是否被引入资源

### 调用方式: GET

### 接口地址:

```
https://api.freelog.com/v2/resources/versions/isCanBeCreate?fileSha1={fileSha1}
```

### url传入参数说明:

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
| fileSha1 | 必选 | string | 文件sha1值 |


### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| [data] | boolean | 是否可以被创建成版本|

### 示例

```js
{
    "ret":0,
    "errcode":0,
    "msg":"success",
    "data":true
}
```
