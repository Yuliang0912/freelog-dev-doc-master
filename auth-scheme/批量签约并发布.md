# 批量签约并发布

### 调用方式: PUT

### 接口地址:

```
https://api.freelog.com/v1/resources/authSchemes/batchSignContracts/{authSchemeId}
```

### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
|  contractId | string | 合同ID |
|  targetId | string | 策略ID(资源Id或presentableId) |
|  resourceId | string | 资源ID |
|  partyOne | int | 甲方 |
|  partyTwo | int | 乙方 |
|  segmentId | string | 策略段落ID |
|  createDate | date | 创建日期 |
|  contractType | int | 合同类型 [[详见附表]][合同类型] |
|  status | int | 合同状态 |
