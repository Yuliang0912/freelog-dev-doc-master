# 创建支付密码


### 调用方式: POST

### 接口地址:

```
https://api.freelog.com/v1/pay/password
```

### body传入参数说明：

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
|password|必选|string|密码,最少6位


### body示例

```js
{
    "password": "123456"
}
```

### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| [data] | bool | 是否成功

### 返回示例

```js
{
    "ret": 0,
    "errcode": 0,
    "msg": "success",
    "data": true
}
```
