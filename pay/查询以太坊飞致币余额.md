# 查询以太坊飞致币余额

### 调用方式: GET

### 接口地址:

```
https://api.freelog.com/v1/pay/helper/feather/balance?address={address}
```

### url传入参数说明：

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
|address|必选|string|以太坊账户地址|

### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| address | string | 以太坊账户地址 |
| balance | string | 飞致币余额(单位:borb) |

### 返回示例

```js
{
	"ret": 0,
	"errcode": 0,
	"msg": "success",
	"data": {
		"address": "0x936DefEF2f4eCa89311473639e78260467b120B9",
		"balance": "0"
	}
}
```
