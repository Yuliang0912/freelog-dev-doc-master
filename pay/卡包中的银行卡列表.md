# 卡包中的银行卡列表

### 调用方式: GET

### 接口地址:

```
https://api.freelog.com/v1/pay/cardclips
```

### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| cardNo | string | 银行好号 |
| currencyType | int | 货币类型 |
| userId | int | 用户ID |
