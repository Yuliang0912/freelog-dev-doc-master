# 更新支付密码

### 调用方式: PUT

### 接口地址:

```
https://api.freelog.com/v1/pay/password/{userId}
```

### url传入参数说明：

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
|userId|必选|int|用户ID

### body传入参数说明：

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
|oldPassword|必选|string|原始密码
|newPassword|必选|string|新密码

### body示例

```js
{
    "oldPassword": "123456789",
    "newPassword": "666666666"
}
```

### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| [data] | bool | 是否成功

### 返回示例

```js
{
    "ret": 0,
    "errcode": 0,
    "msg": "success",
    "data": true
}
```
