# 查询账户余额


### 调用方式: GET

```
https://api.freelog.com/v1/pay/accounts/balance/{accountId}
```

### url传入参数说明：

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
|accountId|必选|string|账户ID

### 返回说明：
| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
|  accountId | string | 账户ID |
|  accountType | int | 账户类型 |
|  balance | string | 余额(最小货币单位),用string类型是因为数字会比较大.超出js整形范围. |

### 返回示例
```js
{
    "ret": 0,
    "errcode": 0,
    "msg": "success",
    "data": {
        "accountId": "feth109b57716a1",
        "accountType": 1,
        "balance": "100000"
    }
}
```
