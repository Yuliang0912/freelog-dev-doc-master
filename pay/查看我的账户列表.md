# 查看我的账户列表

### 调用方式: GET

### 接口地址:

```
https://api.freelog.com/v1/pay/accounts
```

### url传入参数说明：

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
|accountType|可选|int|账户类型,默认全部


### 返回说明：
| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
|  accountId | string | 账户ID
|  accountType | int | 账户类型 [[详见附表]][账户类型] |
|  userId | int | 用户ID
|  balance | decimal | 余额
|  cardNo | string | 绑定的卡号或者eth账户地址
