# 下载存储对象文件

### 调用方式: GET

### 接口地址:

```
https://api.freelog.com/v1/storages/buckets/{bucketName}/objects/{objectName}/file
```

### URL传入参数说明:

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
| bucketName | 必选 | string | bucket名称,1-63个字符之间,数字小写字母和-组成,连接符不能位于头尾 |
| objectName | 必选 | string | 存储对象名称 |

### 返回说明：file-stream
