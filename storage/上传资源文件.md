# 上传资源文件

### 调用方式: POST

### 接口地址:

```
https://api.freelog.com/v1/storages/files/upload
```

### form-data传入参数说明:

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
| resourceType | 可选 | string | 传入此参数,则会进行文件校验检查 |
| file | 必选 | file | 待上传的文件 |

### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| sha1 | string | 存储对象的sha1值 |
| fileSize | number | 文件大小尺寸 |

### 示例

```json
{
    "ret": 0,
    "errcode": 0,
    "msg": "success",
    "data": {
        "sha1": "90db3e14bf81851ee401a02c696a458fbb1f07fc",
        "fileSize": 125123185,
    }
}
```
