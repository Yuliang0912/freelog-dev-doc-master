# 自定义下载用户节点数据


### 调用方式: GET

### 接口地址:

```
https://api.freelog.com/v1/storages/buckets/.UserNodeData/objects/{objectIdOrNodeId}/customPick
```

### url传入参数说明:

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
| objectIdOrNodeId | 必选 | string | 节点对象Id或者节点ID |
| fields | 可选 | string | 指定返回的json对象属性名,多个用逗号分隔 |


### 返回说明：json-file-stream
