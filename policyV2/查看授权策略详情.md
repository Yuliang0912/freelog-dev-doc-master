# 查看授权策略详情

### 调用方式: GET

### 接口地址:

```
https://api.freelog.com/v2/policies/{policyId}
```

### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| policyId | string | 策略ID |
| policyText | string | 策略文本 |
| subjectType | int | 标的物类型 1:资源 2:展品 3:用户组 |
| fsmDescriptionInfo | object | 策略状态机描述 |
| createDate | date | 创建日期 |
