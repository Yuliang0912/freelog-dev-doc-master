# 创建授权策略

### 调用方式: PUT

### 接口地址:

```
https://api.freelog.com/v2/policies/:policyId
```

### body传入参数说明：

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
| policyName | 必选 | string | 策略名称,2-20长度字符 |


### body示例

```js
{
    "policyName":"我的策略"
}

```

### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| [data]] | boolean | 是否修改成功 |

### 返回示例

```js
{
    "ret": 0,
    "errcode": 0,
    "msg": "success",
    "data": true
}
```
