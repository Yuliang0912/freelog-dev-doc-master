# 创建标签

### 调用方式: POST

### 接口地址:

```
https://api.freelog.com/v2/users/tags
```

### body传入参数说明：

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
|tags | 必选 | string[] | tag名称 |
|type | 必选 | int | 来源 1:手动 2:自动 |


### body示例

```json
{
  "tags":["标签1","标签2"],
  "type":1
}
```

### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| tagId | int | 标签ID |
| tag | string | 标签名称 |
| type | int | 标签来源 1:手动 2:自动 |
| totalSetCount | int | 总设置次数 |
