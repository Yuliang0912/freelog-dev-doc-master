# 重置密码



### 调用方式: PUT



### 接口地址:

```
https://api.freelog.com/v2/users/resetPassword
```



### body传入参数说明：

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
|loginName|必选|string|手机号或者邮箱|
|password|必选|string|用户密码,6-24为长度|
|authCode|必选|int|短信或邮件验证码|



### body示例

```json
{
  "loginName":"18923803593",
  "password":"resetPassword#123456",
  "authCode":"471959"
}
```



### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| [data] | bool | 操作结果 |



### 返回示例

```json
{
    "ret": 0,
    "errcode": 0,
    "msg": "success",
    "data": true
}
```
