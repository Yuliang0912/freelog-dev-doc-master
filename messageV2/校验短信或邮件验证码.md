# 发送短信或邮件验证码

### 调用方式: GET

### 接口地址:

```
https://api.freelog.com/v2/messages/verify
```

### url传入参数说明：

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
|authCode|必选|string|验证码|
|address|必选|string|接收验证码的手机号或者邮箱|
|authCodeType|必选|string|验证码类型(register,resetPassword,activateTransactionAccount,updateTransactionAccountPwd)|

### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| [data] | bool | 是否成功 |


### 返回示例

```json
{
    "ret": 0,
    "errcode": 0,
    "msg": "success",
    "data": true
}
```
