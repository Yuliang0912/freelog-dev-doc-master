# 激活个人账户

### 调用方式: PUT

### 接口地址:

```
https://api.freelog.com/v2/accounts/individualAccounts/activate
```

### body传入参数说明:

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
| password | 必选 | string | 支付密码,6位数字组成 |
| authCode | 必选 | string | 验证码 |
| messageAddress | 必选 | string | 验证码发送地址(手机号或邮箱) |

### body示例

```json
{
  "password":"012345",
  "authCode":"123456",
  "messageAddress":"4896819@qq.com"
}
```

### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| [data] | bool | 是否激活成功 |

### 示例

```json
{
    "ret": 0,
    "errcode": 0,
    "msg": "success",
    "data": true
}
```
