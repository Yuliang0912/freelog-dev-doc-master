# 编辑标签信息



### 调用方式: PUT



### 接口地址:

```
https://api.freelog.com/v2/users/tags/{tagId}
```



### url传入参数说明：

| 参数  | 必选 | 类型及范围 | 说明   |
| :---- | :--- | :--------- | :----- |
| tagId | 必选 | int        | 标签ID |





### body传入参数说明：

| 参数 | 必选 | 类型及范围 | 说明     |
| :--- | :--- | :--------- | :------- |
| tag  | 必选 | string     | 标签名称 |



### body示例

```json
{
  "tags":"新的标签名称",
}
```



### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| [data] | bool | 是否操作成功 |



### 返回示例

```json
{
	"ret": 0,
	"errCode": 0,
	"errcode": 0,
	"msg": "success",
	"data": true
}
```
