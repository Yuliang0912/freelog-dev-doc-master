# 删除标签



### 调用方式: DELETE



### 接口地址:

```
https://api.freelog.com/v2/users/tags/{tagId}
```



### url传入参数说明：

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
|tagId|必选|int|标签ID|



### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| [data] | bool | 操作结果 |



### 返回示例

```json
{
    "ret": 0,
    "errcode": 0,
    "msg": "success",
    "data": true
}
```
