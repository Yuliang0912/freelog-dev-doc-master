# 统计展品签约量

> 例如查看当前登录用户消费的展品记录.
>

### 调用方式: GET



### 接口地址:

```
https://api.freelog.com/v2/contracts/subjects/presentables/signStatistics
```



### url传入参数说明:

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
| signUserIdentityType | 必选 | int | 签约方类型(1:甲方 2:乙方) |
| nodeId | 可选 | int | 可选,节点ID |
| keywords| 可选 | string | 搜索关键字 |



### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| subjectId | string | 标的物ID |
| subjectName | string | 标的物名称 |
| policyIds | string[] | 签约的策略 |
| latestSignDate | date | 最后签约日期 |
| count | int | 签约次数 |
