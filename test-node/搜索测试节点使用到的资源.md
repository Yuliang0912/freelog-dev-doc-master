# 搜索测试节点使用到的资源

### 调用方式: GET

### 接口地址:

```
https://api.freelog.com/v1/testNodes/:nodeId/searchTestResourceDependencyTree?keywords={keywords}
```

### 传入参数说明：
| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
| keywords | 必选 | string | 搜索关键字 |


### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| id | string | 实体ID |
| name | string | 实体名称 |
| type | string | 实体类型 |
| versions | string[] | 版本集 |
