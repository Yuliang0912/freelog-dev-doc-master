# 根据测试规则匹配测试结果 (进入测试节点时,调用一次,然后再调用分页查询匹配结果的接口)

### 调用方式: POST

### 接口地址:

```
https://api.freelog.com/v1/testNodes/:nodeId/matchTestResources
```

### 传入参数说明：

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
| nodeId | 必选 | int | 节点ID |


### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| [data] | bool| 是否匹配成功 |


### 返回示例

```js
{
    "ret": 0,
    "errcode": 0,
    "msg": "success",
    "data": true
}
```
