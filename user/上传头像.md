# 上传头像

### 调用方式: POST

### 接口地址:

```
https://api.freelog.com/v1/userinfos/uploadHeadImg
```

### from传入参数说明:

**注意: form表单中file需要放在最后**

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
|file|必选|file|图片资源,不超过2MB,只支持jpg|png|gif格式|

### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| [data] | string | 图片URL|

### 示例

```js
{
    "ret": 0,
    "errcode": 0,
    "msg": "success",
    "data": "https://image.freelog.com/headImage/10022?x-oss-process=style/head-image"
}
```
