# 修改密码


### 调用方式: POST

### 接口地址:

```
https://api.freelog.com/v1/userinfos/updatePassword
```

### body传入参数说明：

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
|oldPassword|必选|string|原始密码|
|newPassword|必选|string|新密码|


### body示例

```json
{
    "oldPassword": "123456",
    "newPassword": "123456abcd"
}
```

### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| [data] | bool | 操作结果 |



### 返回示例

```json
{
    "ret": 0,
    "errcode": 0,
    "msg": "success",
    "data": true
}
```
