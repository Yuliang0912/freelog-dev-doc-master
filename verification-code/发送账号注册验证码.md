# 发送账号注册验证码

### 调用方式: POST

### 接口地址:

```
https://api.freelog.com/v1/message/register
```

### body传入参数说明：

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
|loginName|必选|string|注册的登录名,手机号或者邮箱|
|captchaInput|必选|string| 图片验证码(不区分大小写)|

### body示例

```js
{
    "loginName":"18923803593",
    "captchaInput":"cupb"
}
```

### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| [data] | bool | 是否成功 |


### 返回示例

```js
{
    "ret": 0,
    "errcode": 0,
    "msg": "success",
    "data": true
}
```
