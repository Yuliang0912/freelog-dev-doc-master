# 发送短信验证码

### 调用方式: POST

### 接口地址:

```
https://api.freelog.com/v1/message/send
```

### body传入参数说明：

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
|loginName|必选|string|注册的登录名,手机号或者邮箱|
|authCodeType|必选|string|验证码类型(register,resetPassword)|

### body示例

```js
{
    "loginName":"18923803593",
    "authCodeType":"register"
}
```

### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| [data] | bool | 是否成功 |


### 返回示例

```js
{
    "ret": 0,
    "errcode": 0,
    "msg": "success",
    "data": true
}
```
