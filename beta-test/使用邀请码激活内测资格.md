# 使用邀请码激活内测资格

### 调用方式: POST

### 接口地址:

```
https://api.freelog.com/v1/testQualifications/beta/activate
```

### body传入参数说明：

| 参数 | 必选 | 类型及范围 | 说明 |
| :--- | :--- | :--- | :--- |
| code | 必选 | string | 邀请码(30位长度的字符串) |

### body示例

```js
{
     "code":"OTNkNzQ0YjQtMTlmMS00MTRjLTk3Nz"
}
```

### 返回说明：

| 返回值字段 | 字段类型 | 字段说明 |
| :--- | :--- | :--- |
| [data] | boolean |  申请记录ID |

### 返回示例

```js
{
	"ret": 0,
	"errcode": 0,
	"msg": "success",
	"data": true
}
```
